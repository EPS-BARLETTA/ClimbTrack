❓ Aide

ClimbTrack génère un QR code JSON compatible ScanProf pour centraliser les résultats.

1) S’identifier

Depuis l’accueil, renseigne Nom, Prénom, Classe. Tu arrives ensuite sur le Menu.

2) Choisir une pratique

  • Bloc : choisis une couleur (jaune, vert, bleu, rouge) puis un numéro → “Ajouter à ma liste” pour chaque réussite. “Générer QR” à la fin.
  • Difficulté : indique la voie, la cotation, le relais (X), puis Résultat et Motif si non terminé → “Générer QR”.
  • Vitesse : saisis un temps (mm:ss.cc). Option : Chrono intégré si ton mur n’a pas de chrono.

3) Scanner avec ScanProf

Le QR s’affiche à l’écran. L’enseignant scanne avec ScanProf pour récupérer toutes les infos de la séance.

Clés JSON : {"App":"ClimbTrack","Type":"bloc|difficulte|vitesse","Nom","Prenom","Classe", ...}
